# 🚀 Mayank Sharma — Portfolio

A modern, high-performance developer portfolio built with **React + Vite + TanStack Router**, showcasing projects, skills, and experience with smooth UI and interactive components.

---

## 🌐 Live Demo
👉 https://your-portfolio.vercel.app

---

## ✨ Features

- ⚡ Fast and optimized with Vite
- 🎨 Modern UI with Tailwind CSS
- 🧠 Smooth client-side routing (TanStack Router)
- 🎯 Interactive project showcase
- 📱 Fully responsive design
- 🌙 Clean and minimal dark theme
- 🔥 Animated components and transitions

---

## 🧩 Tech Stack

- **Frontend:** React, TypeScript
- **Build Tool:** Vite
- **Routing:** TanStack Router
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Icons:** Lucide React

## ⚙️ Getting Started

### 1️⃣ Clone the repository


git clone https://github.com/mayank-1584/portfolio.git

cd portfolio


---

### 2️⃣ Install dependencies


npm install


---

### 3️⃣ Run locally


npm run dev


---

### 4️⃣ Build for production


npm run build
